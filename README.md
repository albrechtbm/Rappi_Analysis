# Rappi_Analysis
RappiPlus Business Intelligence Dashboard

End-to-end business analytics project built in Power BI using multiple data sources to evaluate commercial performance, customer behavior, and retention trends.

Project Overview

This project focuses on transforming raw business data into actionable insights through data cleaning, modeling, KPI development, and interactive dashboards.

The analysis includes:

Revenue and profitability performance
Marketing spend efficiency
Customer purchasing behavior
Product and category performance
Cohort analysis and customer retention
Identification of growth opportunities and churn patterns
Key Features
Interactive executive dashboard
Commercial performance analysis
Customer cohort and retention analysis
KPI tracking for revenue, profit, and marketing investment
Data modeling and transformation using Power Query
Business-oriented insights and recommendations
Tools Used
Power BI
Power Query
DAX
Data Modeling
Business Analytics
Business Questions Answered
Which products and categories generate the highest revenue?
Which customer segments contribute the most value?
How effective is marketing investment?
What are the main retention and churn patterns?
Which cohorts show the strongest long-term engagement?
Outcome

The final solution provides stakeholders with a comprehensive view of business performance, enabling data-driven decision making through interactive visualizations and actionable insights.
